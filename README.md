Plateforme de Réservation de Tickets d'Événements


Cette application JavaFX permet la gestion et la réservation de tickets pour différents types d’événements : concerts, spectacles et conférences.
Elle inclut :

Gestion des utilisateurs (Clients et Organisateurs)

Gestion des événements

Gestion des réservations et annulations

Gestion des paiements

Interface graphique simple avec JavaFX
